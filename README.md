# 車両データ管理システム (Fleet Manager)

C言語の学習用として作成した、車両の走行ログ管理・分析ツールです。
メモリ管理（動的確保）、ファイル分割、Makefileによるビルド自動化、安全な文字列操作（snprintf）を実践的に取り入れています。

## 🛠 機能
- **走行データの入力・保存**: リアルタイムな走行データの記録
- **自動分析**: 平均速度、速度超過（100km/h以上）、温度異常（80℃以上）の自動判定
- **高度な検索**: 車両IDによる個別データの抽出と分析
- **ランキング・集計**: 速度順のソート、車両ごとの走行回数集計
- **各種CSV出力**: 全データ出力および速度違反車リストの作成
- **クイックアクセス**: コマンドライン引数による直接機能実行

## 🚀 セットアップ
ビルドには `make` を使用します。

```bash
make
```

##使い方
```bash
./fleet_manager
```

#引数モードについて
```bash
./fleet_manager --ranking      # ランキング表示
./fleet_manager --search 101   # ID 101を検索
./fleet_manager --file log.txt # ログファイルを指定して起動
```
